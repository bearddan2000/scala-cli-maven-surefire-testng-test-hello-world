# scala-cli-maven-surefire-testng-test-hello-world

## Description
A POC for maven app using testNg
framework with surefire plugin.

## Tech stack
- scala
- maven
  - testNg
  - surefire

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
